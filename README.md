# Series 65 Learning Lab

An interactive, single-file study system for the **NASAA Series 65** (Uniform Investment Adviser Law) exam — built to take you from zero financial knowledge to mastery.

### ▶️ Live site: https://pierce-lonergan.github.io/series-65-learning-lab/

## What's inside

- **Study** — 402 flashcards across all four NASAA exam areas; every term is clickable
- **Explore** — a hyperlinked glossary of 320+ concepts written for *zero* prior knowledge, where every definition drills into its sub-concepts (AMT → ISO → bargain element → FMV → …) with a back-stack and breadcrumbs
- **Quiz** — application/calculation questions whose explanations link back into the glossary
- **Exam** — a section-weighted (15 / 25 / 30 / 30), timed simulator with a pass mark of 72% and a review-your-misses screen
- **20+ interactive graphics** — drag a year across MACRS vs ADS depreciation, bend the efficient frontier with a correlation slider, watch a margin call trigger, scrub option payoffs, and more

Each concept includes plain-English definitions, LaTeX-rendered formulas with every variable explained, and worked numeric examples.

## Files

| File | Purpose |
|---|---|
| `index.html` | The interactive Learning Lab (open this) |
| `flashcards.html` | A printable 402-card deck for 3×5 cards (Avery 5388), duplex |

## Print the flashcards

Open `flashcards.html` in Chrome/Edge → `Ctrl+P` → Letter, Portrait, Margins **None**, Scale **100%**, two-sided **Flip on Long Edge**, background graphics **On**.

## Tech

Pure HTML/CSS/JavaScript in a single file — no build step, no dependencies except MathJax (loaded from CDN for equation rendering). Works offline; drop it on a USB stick and it just runs.
